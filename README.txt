
概要

本アプリは USBカメラによる撮像・プレビュー・履歴管理・保存機能 を提供する WPF アプリケーションです。
MVVM アーキテクチャに基づき、疎結合かつテスト可能な設計を意識しています。


実装済み要件

撮像してプレビュー表示

OpenCvSharp を利用し、USBカメラから 1 フレームを取得

プレビュー画面に即時反映

サムネイル履歴（新しい順）

撮像画像をサムネイル化し、一覧表示

選択するとフル画像をプレビュー表示

任意ディレクトリへの保存

「名前を付けて保存」ダイアログから保存先と形式（png/jpg/bmp/tif）を選択可能

必要に応じて OpenCV で再エンコード


実行環境

OS: Windows 10/11

IDE: Visual Studio 2022

フレームワーク: .NET 6 / .NET 7

依存ライブラリ:

 OpenCvSharp4

 Ookii.Dialogs.Wpf

実行方法

Visual Studio でソリューションを開く

NuGet パッケージを復元

USB カメラを接続

F5 で実行

「撮像」ボタン → カメラ撮像 → プレビュー表示 & 履歴追加

「保存」ボタン → ファイルダイアログ → 保存
